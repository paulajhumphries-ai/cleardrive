Mission
Traditional creative software has long been expensive and often inaccessible for many young creators. We wanted to change that. The rapid development of artificial intelligence has introduced a new generation of powerful creative tools, many of which are free to use. This has lowered the barrier to entry for designers, writers, filmmakers and artists around the world in a way that has rarely been discussed.

However, these tools are often difficult to discover and far from common knowledge. ClearDrive was created to solve that problem. ClearDrive is an independent index of free online tools for creatives, bringing together and organising tools across design, typography, writing, photography, video, motion and AI. By making them easier to find, ClearDrive helps creators access the resources they need to experiment, learn and bring their ideas to life.

Website
ClearDrive is a single-page HTML5 website, hand-coded in HTML, CSS, and vanilla JavaScript with no frameworks, libraries, or build tools. The layout uses CSS Grid for the index table and CSS custom properties throughout. Navigation between the landing page, index, and information overlay is handled entirely in JavaScript using scroll-snap and fixed positioning.

Typography
The site uses the native system font stack: -apple-system, BlinkMacSystemFont, Helvetica Neue, Helvetica, Arial, sans-serif — rendering in the default sans-serif of each operating system. On macOS and iOS this is SF Pro, on Windows this is Segoe UI, and on Android this is Roboto. Base font size is 13px with a line-height of 1.4.
